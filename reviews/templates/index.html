A simple Django REST API to create and manage movie reviews.

Admin API: Reviews API: Movies (ping)

Quick API info

  • POST /api/users/register/ — register
  • POST /api/token/ — login (JWT)
  • GET /api/reviews/ — list
  • POST /api/reviews/ — create (auth)
  • POST /api/reviews/<id>/like/ — like/unlike (auth)
Check README.md in repo for setup & Postman collection.
